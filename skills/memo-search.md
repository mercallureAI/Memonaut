# /memo-search — 搜索个人经验知识

## 触发词
`/memo-search <关键词>`

## 核心流程

1. **定位 vault**：读取 `~/.memonaut.json` 获取 `vaultPath`，若不存在则使用 `~/memonaut-vault`
2. **读取索引**：读取 `{vaultPath}/.memonaut/index.md`
3. **关键词匹配**：在索引中搜索用户提供的关键词（匹配标题和标签）
4. **读取匹配文件**：对每个匹配条目，读取对应的 Markdown 文件
5. **返回结果**：展示匹配的经验内容；若无匹配，告知用户未找到

## 输出格式

对每个匹配结果：
```
### [标题]
- 标签: xxx
- 路径: experiences/...

[文件内容]
```

若无匹配：
```
未找到与「关键词」相关的经验记录。
建议：尝试其他关键词，或使用 /memo-save 保存新经验。
```

# /memo-distill — 从情景记忆提炼语义记忆

## 触发词
`/memo-distill [天数|起始月 结束月]`

## 核心流程

1. **定位 vault**：读取 `~/.memonaut.json` 获取 `vaultPath`
2. **确定范围**：
   - 无参数：最近 30 天
   - 数字参数（如 `60`）：最近 N 天
   - 月份参数（如 `2025-01 2025-03`）：指定月份范围
3. **读取情景记忆**：扫描 `memory/episodic/` 对应范围内的文件
4. **识别模式**：分析所有记忆，找出重复出现的主题/模式（≥3次）
5. **生成语义记忆**：对每个识别到的模式，创建或更新 `memory/semantic/[主题]_[类型].md`
6. **展示结果**：告知用户提炼了哪些模式

## 语义记忆模板

```markdown
# [主题] [类型]

- **提炼日期**: YYYY-MM-DD
- **来源数量**: N 条情景记忆

## 模式描述
## 适用场景
## 关键步骤
```
